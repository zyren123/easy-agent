# 01 Linear Layer

This module is the first implementation slice for the curriculum.

## Goal

Explain what a linear layer does, how input and output dimensions relate, and how the
same idea appears in real neural-network libraries.

## Artifacts

- `lesson.py`: readable source-first walkthrough
- `lesson.ipynb`: interactive notebook version
- `src/ai_edu/foundations/linear_layer.py`: shared metadata and future reusable helpers

## Constraints

- Keep the lesson CPU-friendly.
- Avoid introducing a heavyweight training framework in the first slice.
- Keep the notebook aligned with the Python source file.
